# Stocksmith CLI — Homebrew tap

Homebrew formulae for the [Stocksmith CLI](https://github.com/craftybase/stocksmith-cli) —
the command-line interface for the Stocksmith API.

## Install

```sh
brew install craftybase/tap/stocksmith
```

That single command adds this tap and installs the `stocksmith` formula.

## Upgrade

```sh
brew upgrade stocksmith
```

## Uninstall

```sh
brew uninstall stocksmith
```
